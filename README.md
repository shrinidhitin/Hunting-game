Hunting for Your Village is a game inspired by minesweeper


1. Download the program 'hunt_game' in releases
2. In a terminal, open directory with the downloaded item
```bash
cd Downloads
```
3. Run executable
```bash
chmod +x hunt_game
./hunt_game
```
On first launch, macOS may display a security warning. Right click the application and choose Open, or use System Settings → Privacy & Security → Open Anyway.

How to play
- Click on tiles to explore the map.
- Collect:
  - 💰 Gold
  - 🍖 Meat
  - 🏹 Arrows
- If you encounter a 👹 Monster:
  - You automatically use one arrow to defeat it.
  - If you have no arrows remaining, the game ends.
- Gather 10 gold and 10 meat before you can return to the village
